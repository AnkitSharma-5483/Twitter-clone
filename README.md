# X (Twitter) UI Clone

A responsive clone of the **X (formerly Twitter)** homepage built using **HTML and Tailwind CSS**.
This project recreates the dark-themed interface including the sidebar navigation, tweet composer, post feed, trending section, and follow suggestions.

## 🚀 Features

* Responsive layout
* Sidebar navigation with icons
* Tweet composer section
* Post feed UI
* Trending topics panel
* "Who to follow" suggestions
* Sticky navigation bar

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* Google Material Icons

## ▶️ How to Run

1. Clone the repository

2. Open the project folder.

3. Run the `index.html` file in your browser.

## 🎯 Purpose

This project was built to practice **Tailwind CSS, responsive layouts, and modern UI design**.
